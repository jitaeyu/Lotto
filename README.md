# JSP와 JAVA로 만든 로또 추첨 홈페이지 입니다.
#### 로또번호 6자리 + 보너스번호 1자리를 출력해주며 최근 5주간 미 추첨 번호를 표시해 줍니다.
##### 초기 시작 URL은 http://localhost:8686/Lotto/index 입니다.
##### 예시 홈페이지 URL http://yjt3602.cafe24.com/ 입니다.
##### 사용 라이브러리

+ JAVA
  + JSOUP 1.14.3
+ JSP
  + JSTL 1.2.5
+ CSS,JS
  + BootStrap 5.1.3
